# Noetica Ecosystem Bundle (NSC + GLLL + GHLL + GML)
Version: v1.0
Build date: 2026-02-01

This bundle is an integrated, audit-first documentation set for:

- **NSC**: Noetic Compiler + Packet + Bytecode + VM boundary
- **GLLL (Hadamard Glyph LLL)**: low-level ±1 feature-coded glyph layer
- **GHLL (Noetica GHLL)**: high-level meaning / operator language
- **GML (Aeonica Memory Language)**: time + memory + run-ledger braid (PhaseLoom)
- **Compiler Seam**: the single truth bridge between alphabets + ledger provenance

## Core promises (ecosystem-wide)
1. Determinism: same inputs → same decode → same execution trace.
2. Provenance: every lowered artifact retains source spans and a receipt chain.
3. Governance: gates can fail soft/hard with explicit receipt evidence.
4. Layer separation: GLLL ↔ GHLL ↔ GML remain distinct; compilation is the seam.
