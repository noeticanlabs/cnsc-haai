# ATS Summary

**Abstract Theory of Semantics - The Consensus Kernel**

The ATS (Abstract Theory of Semantics) is the core consensus kernel of Coh-AGI. It provides:
- Deterministic state transitions
- Budget-constrained execution
- Receipt-based evidence
