# Glossary

**Coh-AGI Terminology**

| Term | Definition |
|------|------------|
| **ATS** | Abstract Theory of Semantics - the consensus kernel |
| **Coh** | Coherence - the primary optimization objective |
| **JCS** | JSON Canonicalization Scheme (RFC8785) |
| **QFixed** | Fixed-point arithmetic (18 decimal places) |
| **Slab** | Batch of micro receipts with Merkle root |
| **Micro Receipt** | Receipt for a single cognitive step |
| **Chain Hash** | Universal hashing with domain separation |
| **Merkle Root** | Cryptographic commitment to receipt integrity |
| **Fraud Proof** | Dispute proof for invalid receipts |
| **Finalization** | Authorization to delete data after retention |
| **Topology** | Atlas (cognitive state graph) structure |
| **μ (mu)** | Stiffness parameter in (0, 1] |
| **Fatigue** | Accumulated parameter ≥ 0 |
| **Δ_struct** | Change in atlas rank |
| **θ_up/θ_down** | Modulated hysteresis thresholds |

See also: [Main Glossary](../GLOSSARY_AND_LEXICON.md)
